# FLappyfox
Jogo de dsoft
O nome do jogo é flappy fox

os membros do grupo são: Luis Amorim, Joao Klein, Joao peixoto(ele tem faltado).
Para o jogo ser executado você deve possuir a pasta do projeto baixada, e o compilador de python instalado, e clicar em "Run python file" no arquivo "Game.py" 

O botão de interação no flappy fox é a barra de espaço que voce usa para desviar dos obstaculos.
